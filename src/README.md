# Aplicação Petshop

Aplicação de petshop focada no estudo do Angular e do UIKit da plataforma de estudos balta.io.

Funcionalidades básicas na tela: 
- Listar os produtos do petshop
- Fazer cadastro
- Login/Logout
- Alteração de senha
- Incluir e remover produtos do carrinho

Para o backend foram usados dados mocados da aplicação Mockoon (https://mockoon.com/). O arquivo para exportar os dados da api utilizada se encontra na 
pasta assets.

# Petshop

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
